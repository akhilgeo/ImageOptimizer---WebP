mage Optimizer & Converter
A minimalist web-based tool for converting and compressing images to reduce file sizes and improve web loading performance.
Features

Multiple Format Support: Convert PNG, JPG, and GIF images
Output Formats: Export as WebP (best compression) or JPG (universal support)
Batch Processing: Handle multiple images at once
Adjustable Quality: Fine-tune compression with a quality slider (10-100%)
Drag & Drop: Easy file upload via drag-and-drop or file browser
Preserve Filenames: Original filenames are maintained with new extensions
Bulk Download: Download all converted images in a single ZIP file
Real-time Statistics: View original size, compressed size, and savings percentage
Progress Tracking: Visual progress bar during conversion
Client-Side Processing: All processing happens in your browser - no uploads to servers

How to Use
Online Web Tool

Upload Images:

Drag and drop images onto the upload zone, or
Click the upload zone to browse and select files


Choose Settings:

Select output format (WebP or JPG)
Adjust quality slider (higher = better quality, larger file size)


Convert:

Click "Convert Images" button
Wait for processing to complete


Download:

Click "Download ZIP" to get all optimized images
Or start over with new images
